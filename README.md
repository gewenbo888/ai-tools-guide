# AI Tools Guide — AI工具使用指南

[![Live Demo](https://img.shields.io/badge/Live-ai--tools--guide.vercel.app-ff6b35)](https://ai-tools-guide.vercel.app)

A bilingual (Chinese–English) practical guide teaching everyone how to use AI tools across 8 categories. Each tool includes what it does, step-by-step usage instructions, and pro tips.

## Categories

1. **Chat & Reasoning** — ChatGPT, Claude, Gemini, DeepSeek
2. **Image Generation** — Midjourney, DALL-E, Stable Diffusion
3. **Video Generation** — Sora, Kling/Seedance, Runway
4. **Audio & Music** — Suno, ElevenLabs, NotebookLM
5. **Coding** — Claude Code, Cursor, GitHub Copilot
6. **Writing & Design** — Notion AI, Canva AI
7. **Productivity** — Perplexity, Zapier/Make, Gamma
8. **Universal Tips** — 6 principles for mastering any AI tool

## Design

Light, clean editorial aesthetic on warm cream background. Color-coded tool cards with shadow depth, numbered step-by-step guides, General Sans + Noto Sans SC + JetBrains Mono typography, sticky category navigation.

## Tech

Single-page HTML/CSS/JS — no build step, no dependencies.
