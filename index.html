ChatGPT

OpenAI 旗舰对话模型
Conversation 对话
The tool that started the AI revolution. ChatGPT excels at answering questions, writing, brainstorming, analysis, and now multimodal tasks (image, voice, browsing).
引发AI革命的工具。ChatGPT擅长回答问题、写作、头脑风暴、分析，现已支持多模态任务（图像、语音、浏览）。

How to Use 使用方法

  1. Visit chat.openai.com or download the app 访问 chat.openai.com 或下载应用
  2. Type your question or task in natural language 用自然语言输入你的问题或任务
  3. Use GPT-4o for complex reasoning, o3 for deep analysis 复杂推理用GPT-4o，深度分析用o3
  4. Upload images, PDFs, or files for context 上传图片、PDF或文件以提供上下文

Pro Tip 实用技巧

Be specific with your prompts. Instead of "write an email", say "write a professional email to a client declining a meeting, keeping a warm tone, under 100 words."

提示词要具体。不要说"写一封邮件"，而是说"写一封专业邮件婉拒客户的会议邀请，保持温暖语气，100字以内。"

Claude

Anthropic 智能助手
Conversation 对话
Known for nuanced reasoning, long-context understanding (up to 1M tokens), and careful, thoughtful responses. Excellent for analysis, writing, and coding. Claude Code is a powerful CLI coding agent.
以细腻推理、长上下文理解（最高100万token）和深思熟虑的回答著称。擅长分析、写作和编程。Claude Code是强大的命令行编程代理。

How to Use 使用方法

  1. Visit claude.ai or download the desktop/mobile app 访问 claude.ai 或下载桌面/移动应用
  2. Use Projects to organize context and instructions 使用项目功能组织上下文和指令
  3. Upload entire codebases, long documents, or books 上传完整代码库、长文档或书籍
  4. Install Claude Code (CLI) for coding: npm i -g @anthropic-ai/claude-code 安装 Claude Code (命令行) 进行编程

Pro Tip 实用技巧

Claude's long context window is its superpower. Upload an entire repository, a 300-page PDF, or a full book and ask questions about it — it remembers everything.

Claude的长上下文窗口是其超能力。上传整个代码仓库、300页的PDF或一整本书并提问——它能记住所有内容。

Google的多模态AI，集成于搜索、Gmail、文档和Android。擅长实时信息、Google Workspace整合和多模态理解。

How to Use 使用方法

  1. Visit gemini.google.com or use the Gemini app 访问 gemini.google.com 或使用Gemini应用
  2. Activate in Google Docs/Sheets/Slides for inline help 在Google文档/表格/幻灯片中激活获取内联帮助

DeepSeek

深度求索
Reasoning 推理
China's open-source reasoning powerhouse. DeepSeek-R1 shows its chain-of-thought reasoning process, making it excellent for math, logic, and coding problems. Free and open-weight.
中国的开源推理引擎。DeepSeek-R1展示思维链推理过程，特别擅长数学、逻辑和编程问题。免费且开源权重。

How to Use 使用方法

  1. Visit chat.deepseek.com (free, no login required for basic use) 访问 chat.deepseek.com（基础使用免费无需登录）

AI Image Tools

AI图像工具 — 文字变图画
🎨

Midjourney

最强AI绘画工具
Image 图像
The gold standard for AI art. Produces stunning, highly aesthetic images with a distinctive artistic style. Best for concept art, illustrations, and creative visuals.
AI艺术的黄金标准。生成令人惊叹的高美感图像，具有独特的艺术风格。最适合概念艺术、插画和创意视觉。

How to Use 使用方法

  1. Join Discord or use midjourney.com (web editor) 加入Discord或使用 midjourney.com（网页编辑器）
  2. Type /imagine followed by your text prompt 输入 /imagine 加上你的文字描述
  3. Use --ar 16:9 for widescreen, --v 7 for latest model 用 --ar 16:9 设宽屏，--v 7 用最新模型
  4. Upscale (U) or create variations (V) of results 放大(U)或创建变体(V)

Pro Tip 实用技巧

Add style keywords: "cinematic lighting, 8k, detailed, artstation" for photorealistic results. For illustration: "watercolor, studio ghibli, soft palette".

添加风格关键词："cinematic lighting, 8k, detailed"获得逼真效果。插画风格用"watercolor, studio ghibli, soft palette"。

OpenAI 图像生成
Image 图像
Integrated directly into ChatGPT. Best for quick image generation during conversations, editing existing images, and generating images with accurate text rendering.
直接集成在ChatGPT中。最适合对话中快速生成图像、编辑现有图像和生成包含准确文字的图像。

How to Use 使用方法

  1. Open ChatGPT and simply ask it to create an image 打开ChatGPT直接要求它创建图像
  2. Describe what you want in natural language 用自然语言描述你想要的内容
  3. Ask for edits: "make the background blue" or "remove the person" 要求编辑："把背景改成蓝色"或"移除人物"

开源图像生成
Open Source 开源
The open-source champion. Run locally on your GPU for unlimited free generation. Massive community of models, LoRAs, and ControlNet extensions. Full creative control.
开源冠军。在本地GPU上运行，无限免费生成。拥有庞大的模型社区、LoRA和ControlNet扩展。完全的创作控制。

How to Use 使用方法

  1. Install ComfyUI or AUTOMATIC1111 WebUI locally 本地安装 ComfyUI 或 AUTOMATIC1111 WebUI
  2. Download models from CivitAI or HuggingFace 从 CivitAI 或 HuggingFace 下载模型
  3. Or use cloud: stability.ai, RunDiffusion, or Replicate 或使用云端：stability.ai、RunDiffusion 或 Replicate

Universal Tips 通用技巧

Master Any AI Tool

掌握任何AI工具的通用法则

Be Specific, Not Vague

具体，不要模糊

"Write a 200-word product description for a wireless earphone targeting fitness enthusiasts, emphasizing sweat resistance and bass quality."

"为无线耳机写一段200字的产品描述，目标用户是健身爱好者，强调防汗和低音品质。"

Give Examples

提供示例

Show the AI what you want by providing examples of the style, format, or tone you're looking for. "Write it like this example: [paste example]."

通过提供风格、格式或语气的示例来展示你想要什么。"按照这个例子来写：[粘贴示例]。"

Iterate, Don't Start Over

迭代，不要重来

If the first result isn't perfect, refine it: "Make it shorter", "More formal", "Add humor". Each round gets closer to what you want.

如果第一次结果不完美，继续改进："缩短一点"、"更正式"、"加点幽默"。每一轮都更接近你想要的。

Assign a Role

分配角色

"You are a senior marketing strategist. Review my campaign plan and suggest improvements." Role-playing dramatically improves output quality.

"你是一位资深营销策略师。审查我的活动方案并提出改进建议。"角色扮演能显著提升输出质量。

Use the Right Tool

用对工具

ChatGPT for conversation. Claude for analysis. Midjourney for art. Perplexity for research. Don't use a hammer when you need a screwdriver.

ChatGPT用于对话，Claude用于分析，Midjourney用于艺术，Perplexity用于研究。需要螺丝刀的时候别用锤子。

Always Verify

始终验证

AI can hallucinate. Always fact-check critical information, especially numbers, dates, quotes, and legal/medical advice. Trust but verify.

AI会产生幻觉。始终核查关键信息，特别是数字、日期、引用和法律/医学建议。信任但要验证。
